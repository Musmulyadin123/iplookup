# Simple-IP-Lookup

This Python3 Script simply write out info about ip that you give in to!

## Lookup Features
```
Features: ISP
          Organisation
          City
          Region name
          Coordinates
          AS Number
          Zip
          Proxy or not
          Reverse lookup
          Hosting or not
          
```

## Your informations Features
```
Features: Your public IPV4
          Your public IPV6
          Hostname
          Your Country
```
